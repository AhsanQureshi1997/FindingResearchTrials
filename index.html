UACOMP Active Trials Dashboard

This prototype pulls active studies from the public ClinicalTrials.gov API and tries to approximate UACOMP-related activity by searching across multiple institutional aliases. Because clinical trial records are not always tagged to the medical school using one exact name, the best version of this tool searches several UArizona and Banner-Phoenix variants, then de-duplicates by NCT ID.
Tip: add investigator names, departments, or center names if you want broader recall.
Source endpoint: ClinicalTrials.gov public API.
Method: query multiple aliases, merge all returned studies, then deduplicate by NCT ID.
Best use: admissions-facing proof of concept, faculty outreach, trial landscape scans, and identifying investigators or departments with live study activity.
